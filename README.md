# lesson-python

## 環境構築

**下記のインストール**
- VSCode
- Docker

**VSCodeの下記の拡張機能のインストール**
- Remote - Containers

## 環境起動
- Docker Desktopの起動
- VSCodeの起動
- VSCode左下の青いRemoteアイコンを押下
- `Open folder in Container` を選択
- `lesson-python` を選択
